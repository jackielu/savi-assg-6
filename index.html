About this map

This map was created for XSAVI-780 Assignment 6. It is an update of Assignment 5 using the Skeleton boilerplate

Data preparation involved downloading the KML data from the New York State GIS Clearinghouse, and using geojson.io to convert the data to GeoJSON format. Also had to use QGIS to convert data from singlepart polygons to multipart polygons

Data source: New York State Ecozones / Ecological Regions, after Will et al. (1982) and Dickinson (1983)

Published by New York State DEC, July 3rd 2008.

Feature labels were created using the plug-in Leaflet.label
